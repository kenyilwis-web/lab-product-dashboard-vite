# Product Dashboard

## Project Description

This project is a React-based product dashboard for an e-commerce workflow. It renders product cards dynamically, supports filtering by stock availability, applies conditional styling for out-of-stock products, and includes a remove action per product card.

## Setup Instructions

1. Clone the repository:
   ```sh
   git clone https://github.com/kenyilwis-web/lab-product-dashboard-vite.git
   cd lab-product-dashboard-vite-tests
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Running Tests

Run the full test suite:

```sh
npm test
```

Run tests once in non-watch mode:

```sh
npx vitest run
```
